## Architecture overview

* Router: [github.com/gorilla/mux](github.com/gorilla/mux)
* ORM : [https://github.com/jinzhu/gorm](https://github.com/jinzhu/gorm)
* WebSockets: [golang.org/x/net/websocket](golang.org/x/net/websocket)
* Database: [sqlite3](github.com/mattn/go-sqlite3)
* JWT Auth: [jwt-go](https://github.com/dgrijalva/jwt-go)
